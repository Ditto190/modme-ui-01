# VT Code Bridge

Enables deep integration with VT Code for file manipulation and shell execution.

## Tools

### `edit_file`
Semantic file editing via VT Code.

### `search_symbols`
Code symbol search using Tree-sitter.

### `create_pty_session`
Opens a new terminal session in the IDE.

## Configuration
Requires `VTCODE_MCP_URL` environment variable.
