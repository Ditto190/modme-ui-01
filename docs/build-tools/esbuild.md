# ESBuild Integration

**Category**: build-tools
**Status**: 🟢 Active
**Last Updated**: 

## Summary

TypeScript/React build configuration with ESBuild for fast compilation

## Quick Start

**build**: `npm run build:esbuild`
**watch**: `npm run watch:esbuild`
**type_check**: `npx tsc --noEmit`

## Key Concepts

### setup

Single package.json script for fast builds

### configuration

esbuild.config.mjs with React/TS presets

### benefits

10x faster than tsc, sub-second rebuilds

### trade_offs

No type checking (use tsc separately)
