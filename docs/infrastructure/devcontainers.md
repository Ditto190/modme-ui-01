**Category**: infrastructure

## Summary

VS Code Dev Containers for consistent Linux development environment on Windows

## Quick Start

**build**: `F1 → Dev Containers: Rebuild Container`
**attach**: `F1 → Dev Containers: Attach to Running Container`
**logs**: `docker logs vsc-modme-ui-01-test-worktree`

## Key Concepts

### purpose

Cross-platform development consistency

### features

Python, Node.js, Git, MCP servers pre-configured

### setup_time

5-10 minutes first run, instant subsequent starts

### benefits

Identical environment for all developers
