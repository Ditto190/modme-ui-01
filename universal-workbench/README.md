# Universal Workbench

Welcome to your **Universal Workbench**! This environment is designed to be your go-to place for editing code, experimenting with tools, and learning, compatible with GitHub Codespaces.

### In GitHub Codespaces

1. Push this folder to a GitHub repository.
2. Click the **Code** button on GitHub and select **Codespaces**.
3. Create a new codespace. It will automatically load with all the tools you need!

### In VS Code Desktop

1. Make sure you have Docker installed and the "Dev Containers" extension.
2. Open this folder.
3. Click the pop-up (or press F1 and search) to **"Reopen in Container"**.
